# The Tide Playbook 🦀

A working playbook site for **Blue Crab Strategies** — built for a funder working session
during **London Climate Week** — on shaping climate narratives through the media and culture
people already love.

## What's inside

- **The case for story** — the data on climate's near-total absence from entertainment
  (2.8% of 37k+ scripts mention any climate keyword) and why that whitespace is a
  philanthropic opportunity.
- **The TIDE framework** — Tune in · Immerse · Distribute · Evaluate: four moves that
  work in any medium.
- **Nine case files** — Grey's Anatomy's heat-dome arc, the UK soaps' COP26 crossover,
  the Sabido telenovela method, *Don't Look Up*, K-drama whitespace + Kpop4Planet,
  Coldplay's MIT-verified tour, Billie Eilish's Overheated at The O2, Playing for the
  Planet, and Forest Green Rovers — each with a "steal this" takeaway.
- **The play builder** — an interactive tool that drafts an opening play from any
  medium × goal combination, ready to copy and pressure-test in the room.
- **In the room** — a 90-minute working-session arc for the London convening.

## Running it

No build step. Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

Static HTML + CSS + vanilla JS; fonts load from Google Fonts. The print stylesheet
turns the page into a session handout.
